# Software Factory: Ultimate AI Builder Platform

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Software Factory** is a Meta-Orchestrator designed specifically to run alongside autonomous systems like Google Antigravity. It forces AI agents into strict Planner/Builder loops to systematically scaffold extremely complex codebases without context degradation.

## Features

- **Deterministic Planning:** Forces AIs to strictly separate "Planning" from "Building".
- **Dynamic Prompt-OS:** Extracts the best prompt DNA from industry leaders (Claude Code, Cursor, Devin) to maximize instruction adherence.
- **Native Antigravity Integration:** Includes a `.md` workflow file to instruct Google Antigravity to operate as the engine natively.

## Installation & Usage

### The Easiest Way: Global Installation

You can install the CLI globally using `uv` (or `pipx`) to easily initialize the Software Factory brain in any new project.

```bash
uv tool install git+https://github.com/eybersjp/software-factory.git
```

Once installed globally, simply navigate to your new project folder and run:

```bash
software-factory init
```

This will automatically generate the required `.agents/workflows/SF-software-factory.md` file. You can then open Antigravity and type:

```bash
/SF-software-factory Build me a scalable full-stack React / FastAPI application.
```

### Advanced: Standalone Python CLI

If you want to bypass the Antigravity chat and use the framework to output scaffolding plans entirely in a terminal:

```bash
uv pip install -e .
software-factory scaffold-app "A Python FastAPI backend"
```
